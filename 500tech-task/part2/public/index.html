The Challenge

Implement auto-expandable input control which meets the following requirements:

  1. Should be expanded when the user hovers above it
  2. Should remain expanded when the user focuses inside and starts typing
  3. Should collapse if the user leaves the field empty
  4. Should stay expanded as long as there is a value inside of it
